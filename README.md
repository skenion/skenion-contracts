# Skenion Contracts

Versioned contracts, schemas, generated protocol packages, and conformance tests for Skenion. This repository is the source of truth for TypeScript/Rust communication contracts.

## Status

Bootstrap repository for the Skenion project. Implementation will follow the public architecture and release rules defined in [EchoVisionLab/skenion](https://github.com/echovisionlab/skenion).
